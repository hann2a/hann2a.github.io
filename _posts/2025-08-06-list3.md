## 2차원 배열 
### 2차원 배열의 선언 

- 1차원 리스트를 묶어 놓은 리스트 

- 2차원 이상의 다차원 리스트는 차원에 따라 인덱스를 선언

- 2차원 리스트의 선언: 세로 길이(행의 개수), 가로길이(열의 개수)를 필요로 함

- 파이썬에서는 데이터 초기화를 통해 변수 선언과 초기화가 가능함 

`arr = [[0, 1, 2, 3], [4, 5, 6, 7]] (2행 4열의 2차원 리스트)`

| arr |0  |1  | 2 | 3 | 열 |
| --- |---|---|---|---|---|
| 0 |0 |1|2|3||
| 1 |4 |5|6|7||
| 행 | |||||

### 2차원 배열 순회 
순회: 배열의 모든 요소에 한 번씩 접근하는 것 

```python
# 예시: 1부터 9까지 숫자를 3x3 형태로 담은 리스트

arr = [
    [1, 2, 3], 
    [4, 5, 6], 
    [7, 8, 9],
]

# 행의 수
N = len(arr)

# 열의 수
M = len(arr[0])
```
#### 행 우선 순회 

가장 일반적인 순회 방식으로, 바깥쪽 반복문이 행을, 안쪽 반복문이 열을 담당합니다. 
```python
for i in range(N):
  for j in range(M):
    print(arr[i][j], end =' ')
```
<br>

역-행 우선 순회 
```python
for i in range(N - 1, -1, -1):
    for j in range(M):
        print(arr[i][j], end=' ')
```

#### 열 우선 순회 

바깥쪽 반복문이 열을, 안쪽 반복문이 행을 담당 

```python
for j in range(M):
  for i in range(N):
    print(arr[i][j])
```
역-열 우선 순회 

### 델타 이동 
2차원 배열의 특정 위치에서 상하좌우 등 인접한 칸을 탐색할 때 매우 유용한 기법 

#### 델타 이동 활용 예시 
arr의 중앙값 5의 위치 (1, 1)에서 상하좌우 값을 출력해 봅시다.

- 파이썬은 음수 인덱스를 써서 주의해야 함 

```python
arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# 상, 하, 좌, 우 순서로 방향 벡터(델타) 정의

# r(행)의 변화량
dr = [-1, 1, 0, 0]

# c(열)의 변화량
dc = [0, 0, -1, 1]

# 기준점 
r = 1
c = 1

# 행, 열 값
N = len(arr)
M = len(arr[0])

# 4방향 확인 -> 총 반복도 4번 
for i in range(4):
  # 새로운 위치 이동 
  # dr과 dc의 인덱스 값이 곧 방향이 된다. 
  nr = r + dr[i]
  nc = c + dc[i]
  
  # 이동하고 나서 확인해야 할 것이 있음 
  # 경계(벽)를 체크 
  # 이동 후 위치가 배열의 범위를 벗어나지 않는지를 확인 
  # 1. 범위 안의 좌표만 처리하기 ------------------------------------
  if 0 <= nr < N and 0 <= nc < M: 
    # 이동한 위치의 값을 출력 or 유효한 좌표에 대한 로직을 작성 
    print(arr[nr][nc])
  # 2. 범위를 벗어나는 좌표는 건너뛰기 ------------------------------------
  if nr <0 or nr >= N or nc < 0 or nc >= M:
    continue
  # 이 아래는 모두 유효한 좌표임이 보장됨 
  print(arr[nr][nc])
```

### 연습문제 _1_Delta
```python
T =int(input())

dr = [-1, -1, 0, 0]
dc = [0, 0, -1, 1]

for tc in range(1, T+1):
  # N 값 설정 (행, 열 길이)
  N = int(input())

  # 이차원 배열 설정 
  arr = [list(map(int, input().split())) for _ in range(N)]

  # 모든 요소의 차이 절대값을 누적시킬 최종 값
  result = 0 
  # 2차원 모든 요소 순회 
  # r는 행의 인덱스 c는 열의 인덱스 
  # 첫 for문 두 개는 이동을 위한 for문
  for r in range(N):
    for c in range(N):
      # 탐색을 위한 반복 
      # 현재 위치 (r, c)의 상하좌우 인접한 요소를 확인하기 위한 반복 
      for i in range(4):
        nr = r + dr[i]
        nc = c + dc[i]

        # 인접한 요소의 좌표가 벽이 아닐 때만 계산 진행 
        if 0 <= nr < N and 0 <= nc < N:
          # 현재 칸과 인접한 칸의 값의 차이를 구해서 절댓값을 계산
            result += abs(arr[nr][nc] - arr[r][c])
```

```python
# 테스트 케이스 개수
T = int(input())
for test_case in range(1, T+1):

  # 영역의 개수
  count = int(input())

  # f_r, f_c : 왼쪽 위 모서리
  # s_r, s_c : 오른쪽 아래 모서리
  # color : 해당 칸이 어떤 색인지

  # 왼쪽 위 모서리와 오른쪽 아래 모서리 정보
  #  -> 해당 영역의 인덱스 묶음을 튜플로 저장하여 모든 리스트 반환

  def find_index(f_r, f_c, s_r, s_c):
    box = []
    for i in range(f_r, s_r + 1):
      for j in range(f_c, s_c + 1):
        box.append((i, j))
    return box


  # 빨간 박스의 인덱스 모으기
  red_box_result = []
  # 파란 박스의 인덱스 모으기
  blue_box_result = []

  for _ in range(count):
  # 박스 한개씩 출력
    f_r, f_c, s_r, s_c, color = list(map(int, input().split()))

    # color가 빨간색이면 해당 칸에 해당하는 좌표 모두를 튜플로 묶어서 넣기
    # 리스트 속 튜플로 담기므로 하나씩 꺼내서 다시 넣기 
    if color == 1:
        red_box = find_index(f_r, f_c, s_r, s_c) # 빨간 박스의 인덱스 담김
      # 빨간 박스 통합
        for red in red_box:
            red_box_result.append(red)
    else:
        blue_box = find_index(f_r, f_c, s_r, s_c) # 파란 박스의 인덱스 담김
        # 파란 박스 통합
        for blue in blue_box:
            blue_box_result.append(blue)
  # 여기까지 인덱스 모음 -------------------------------------
  # 이 아래부터 보라색 박스
  # 집합으로 바꿔서 중복 제거, 교집합을 통해 보라색 위치 찾기 
  # len을 통해 개수 세기 
  purple_box = len(set(red_box_result) & set(blue_box_result))
```

```python
"""
N : 행의 개수 // 3<=N
M : 열의 개수 // M<=100

# 테스트케이스 
T = int(input())

# 델타 검색을 위한 배열 만들기 
def make_delta(n):
  dr = [-n, n, 0, 0]
  dc = [0, 0, -n, n]
  return dr, dc 

for test_case in range(1, T+1):
  # 행과 열의 개수 받기 
  N, M = map(int, input().split())
  # 배열 받기 
  arr = [list(map(int, input().split())) for _ in range(N)]
  
  # 1. 원래 행렬은 arr 

  # 2. 해당 칸의 값만큼 상하좌우로 값을 구하는 것 
  # 합산 값 구하는 리스트 
  add_everything = []
  for r in range(N):
    for c in range(M):
        # make_delta에 현재 칸의 값을 넣어서 이동 가능한 로직 만들기  
        dr, dc = make_delta(arr[r][c])
        # 상하 
        nr_l = r + dr[0]
        nr_r = r + dr[1]
        # 좌우
        nc_u = c + dc[2]
        nc_d = c + dc[3]

         # 범위를 판 안으로 잘라서 합산
        row_start = max(0, nr_l)
        row_end   = min(N - 1, nr_r)
        col_start = max(0, nc_u)
        col_end   = min(M - 1, nc_d)

        width_sum  = sum(arr[i][c]   for i in range(row_start, row_end + 1))
        height_sum = sum(arr[r][j]   for j in range(col_start, col_end + 1))
        result_sum = width_sum + height_sum - arr[r][c]
        add_everything.append(result_sum)

  # 3. 최대값 비교하기 
  max_result = max(add_everything)
  print(f"#{test_case} {max_result}")
```
