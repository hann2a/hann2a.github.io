## 배열 Array

일정한 자료형의 변수들을 하나의 이름으로 **열거**하여 사용하는 자료구조 

### 배열의 필요성 

- 여러 개의 변수가 필요할 때 일일이 다른 변수명을 이용하는 것은 매우 비효율적 

### 1차원 배열 

append가 될 때마다 칸이 추가되는 게 아니라 파이썬 자체적으로 1.5배 정도 추가하는 것 

배열의 크기만큼 복사하는 시간이 추가로 걸림 

1차원 배열 선언의 예시 
<br>

```python
arr = list()
arr = []
arr = [0] * 10
arr = [1, 2, 3]
```

### 입력 받은 정수를 1차원 배열에 저장하는 방법 

```python
N = int(input())
arr = list(map(int, input().split()))
```

### 배열 원소의 합을 계산하는 코드

```python
s = 0
for i in range(N):
  s += arr[i]
```

```python 
for x in arr:
  s += x
``` 

### 배열 원소의 최댓값을 찾는 코드 
```python 
max_v = arr[0] # 첫 원소를 최대로 가정 
for i in range(1, N): # 첫 원소 다음부터 세기 2~N
  if max_v < arr[i]:
    max_v = arr[i] # 돌리는 원소가 현재 최댓값보다 크면 재할당 


```

### 배열 원소 중 최댓값의 인덱스 max_idx 찾기 

```python
max_idx = 0
for i in range(1, N):
  if arr[max_idx] < arr[i]:
    max_idx = i 
```

```python
max_idx = 0 # 일단 최댓값의 인덱스가 0이라고 가정하고 시작 
for i in range(1, N): # 다음 인덱스부터 조사 
  if arr[max_idx] < arr[i]: # 현재 인덱스의 값보다 돌아가는 인덱스에 해당하는 값이 더 크다면
    max_idx = i # 지금 돌아가는 인덱스로 최대값의 인덱스를 재배정 
```

### 최댓값이 여러 개인 경우 마지막 인덱스 max_idx 찾기 

```python
max_idx = 0 #첫 원소를 최대로 가정 
for i in range(1, N):
  if arr[max_idx] <= arr[i]:
    max_idx = i 
```

```python 
max_idx = 0 # 첫 원소를 최대로 가정 
for i in range(1, N):
  if arr[max_idx] <= arr[i]: # 같거나 크면 i 재배정(그러면 갱신됨)
    max_idx = i 
```

### 찾는 값이 배열에 있으면 해당 원소의 인덱스, 없으면 -1을 idx에 넣기 

입력 예시 
```
6 5
2 7 5 3 1 7
```
<br>
정수 N을 입력받고, 공백으로 구분된 정수들을 리스트로 입력받는 코드입니다. 
<br>

```python 
N, V = map(int, input().split()) # N, 찾는 값 V
arr = list(map(int, input().split()))

idx = -1 # 찾는 값이 없다고 가정 
for i in range(N):
  if arr[i] == V: # arr[i]가 찾는 값이면 
    idx = i # 인덱스 저장 
    break # for i 
```

## 정렬 

2개 이상의 자료를 키(특정 기준)에 의해 작은 값부터 큰 값(오름차순: ascending), 혹은 그 반대의 순서대로(내림차순: descending) 재배열하는 알고리즘 

### 버블 정렬 

인접한 두 개의 원소를 비교하며 자리를 계속 교환하는 방식 

### 버블 정렬 과정 

1. 첫 번째 원소부터 인접한 원소끼리 계속 자리를 교환하면서 맨 마지막 자리까지 이동 
  
2. 한 단계가 끝나면 가장 큰 원소가 마지막 자리로 정렬된다. 

3. 교환하며 자리를 이동하는 모습이 물 위에 올라오는 거품 모양과 같다고 하여 버블 정렬이라고 함

### 시간 복잡도 
```latex
$$O(n^2)$$
```
<br>

```
BubbleSort(a, N) # 정렬할 배열과 배열의 크기 
  for i: N-1 -> 1 # 정렬할 구간의 끝 
    for j: 0 -> i -1 # 비교할 원소 중 왼쪽 원소의 인덱스 
     if a[j] > a[j+1] # 왼쪽 원소가 더 크면 
        a[j] <-> a[j+1] # 오른쪽 원소와 교환 
```
<br>

```python 
def bubble_sort(a, N): # 정렬할 List, N 원소 수 
  for i in range(N-1, 0, -1): # 구간의 끝 i: N-1 -> 1 (범위의 끝 위치)
    for j in range(i): # 비교할 왼쪽 원소 인덱스 j 
      if a[j] > a[j+1]:
        a[j], a[j+1] = a[j+1], a[j]

arr = [55, 7, 78, 12, 42]
bubble_sort(arr, len(arr))
print(arr)
```

for test_case in range(1, T+1):
    N = int(input()) # 가로 길이
    height = list(map(int, input().split())) # 7 4 2 0 0 6 0 7 0
    difference_ = [] # 낙차 저장
    
    # 모눈 공간은 N * max(height)
    for i in range(N):  # 가로 기준 블럭을 돈다
        diff_ = sum(1 for n in height[i + 1:] if height[i] > n)  # 내 기준 내 오른쪽에 있는 것들 중에 나보다 작은 거를 세서 더함
        difference_.append(diff_) # 낙차 리스트에 추가 

    max_result = max(difference_) # 낙차 중에 제일 큰 거를 고른다 
    print(f"#{test_case} {max_result}")
