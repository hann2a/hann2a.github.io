## Daily Review
`Python -i`
터미널에서 이 명령어로 파이썬을 실행해볼 수 있다. 

### Python 기본 

### 표현식 Expression

하나의 값으로 평가될 수 있는 모든 코드 

### 값 Value

더 이상 계산되거나 평가될 수 없는, 프로그램의 가장 기본적인 데이터 조각 

### 변수 Variable 

  - 값을 나중에 다시 사용하기 위해, 그 값에 붙여주는 고유한 이름 
  - 변수는 메모리 주소를 포함하지 않고 가리킨다. (refer/point to)

### 변수 할당 Variable Assignment

- 표현식이 만들어낸 값에 이름을 붙이는 과정(연결)
- `=` 할당 연산자: 오른쪽 표현식의 평가 결과 값을 왼쪽 변수에 저장 

### 메모리 주소

특정 데이터 값을 정확히 찾아가기 위해 사용하는 기계적인 숫자 주소 

### 객체 Object

값 + 타입 + 주소 정보를 묶은 것을 객체라고 부른다. 

### 타입 Type

값, 값에 적용할 수 있는 연산 

## Type

### 숫자형 데이터 Numeric Types

정수 자료형(int)   : 소수점이 없는 숫자를 표현 
실수 자료형(float) : 소수점이 있는 더 정밀한 숫자를 표현 

### 지수 표현법
```
# 1,230,000,000 (1.23 * 10^9)
big_number = 1.23e9

#0.00314 (3.14 * 10^-3)
small_number = 3.14e-3
```

### 연산자 우선순위

- 높음: 지수, 음수 부호, 곱셈, 나눗셈, 정수 나눗셈, 나머지
- 낮음: 덧셈, 뺄셈

```
# 2진수 10은 10진수로 2입니다 (1 * 2^1 + 0 * 2^0)
print(0b10)

# 8진수 30은 10진수로 24입니다. (3 * 8^1 + 0 * 8^0)
print(0o30)

# 16진수 10은 10진수로 16입니다. (1 * 16^1 + 0* 16^0)
print(0x10)
```
어떤 진법으로 코드를 작성하든, 파이썬은 내부적으로 모두 같은 숫자 값으로 인식하며, 

print()함수는 그 값을 우리에게 익숙한 10진수로 변환하여 보여준다.
